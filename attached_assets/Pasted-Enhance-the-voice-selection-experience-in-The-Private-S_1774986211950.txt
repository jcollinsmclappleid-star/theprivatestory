Enhance the voice selection experience in The Private Story so it feels more personal, premium, and emotionally distinctive.

This is not just a “voice settings” panel.
It should feel like the user is choosing who is telling their story.

The current structure is good, but it feels too functional and abstract.
We need to make it feel:
- more human
- more memorable
- more emotionally engaging
- more premium
- more aligned with the luxury, female-first, private brand

==================================================
OBJECTIVE
==================================================

Transform the voice selector from a technical choice into a subtle narrator identity choice.

The user should feel like they are choosing:
- a presence
- a tone
- a storyteller

Not just:
- a voice preset

Keep the design restrained and elegant.
Do not make this feel playful, cartoonish, or like a marketplace.

==================================================
CORE IMPLEMENTATION RULES
==================================================

1. Add a first name to each voice
2. Keep the style label
3. Add a small abstract visual identity per voice
4. Improve the copy so it feels like a person, not a setting
5. Add “Best for” guidance under each description
6. Keep Jane / Classic as the default and recommended voice
7. Keep UI luxury, minimal, and calm

Do NOT use photography.

- abstract feminine silhouette
- minimal line-art profile
- soft gradient shape
- blurred portrait-style abstract treatment
- elegant, premium, low-detail visuals

Each voice should feel visually distinct, but still part of one cohesive brand system.
