Build MoonlitAudio as an AI-powered personalised audio story product using a minimal user intake and a hidden story-planning layer behind the scenes.

IMPORTANT PRODUCT LOGIC:
Do NOT generate the final story directly from raw user input.
Instead use this pipeline:

1. User fills minimal intake
2. Backend creates hidden structured story brief
3. Backend writes final story from that brief
4. Backend generates one cover image + one image per scene
5. Backend generates one narrated audio file
6. Frontend plays audio while visuals change by scene progress

9. SCENE SYNC LOGIC
==================================================

During playback:
- determine current progress
- map progress to current scene
- update displayed scene image

Pseudo logic:

const progress = audio.currentTime / audio.duration;
const sceneIndex = Math.floor(progress * story.scenes.length);
currentImage.src = images.scenes[sceneIndex].url;

Add smooth fade transition between images.
